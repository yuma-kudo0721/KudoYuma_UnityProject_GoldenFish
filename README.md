# 金魚すくいゲーム

## 概要
本作は、魚をポイですくうことで制限時間を延ばし、ハイスコアを目指すアクションゲームです。  
さまざまな種類の魚が登場し、それぞれ異なる行動や特徴を持っています。

ゲーム中に FEVERゲージ を貯めることで、ポイが手に変化し、魚を捕り逃した際のクールダウンが発生しなくなります。  
これにより、一気にスコアを伸ばす爽快なプレイ体験を実現しています。

また、オンラインランキング機能を実装しており、他のプレイヤーとスコアを競うことが可能です。  
友人や知り合いにも実際にプレイしてもらい、楽しんでもらうことができました。

## 遊び方
1. タイトル画面で名前を入力  
2. ポイを使って魚を捕る  
3. FEVERゲージを貯め、より多くの魚を捕って制限時間を延ばす  

## 得点を伸ばすコツ
1. ポイで魚を獲り逃すとクールダウンが発生するため、慎重に狙う  
2. FEVER状態（手になった状態）では、とにかく連打して魚を捕る  
3. 妨害行動を行う魚を優先的に捕る

## プログラム面（こだわり）
- 魚キャラクターの共通挙動をクラス設計で管理  
  - 共通処理をまとめることで、新しい魚キャラクターの追加を容易に  
  - コードの重複を防ぎ、保守性を向上  
  - 挙動の一貫性を保ったまま拡張が可能  
  - 魚の削除は画面外で行い、見た目の違和感を防止  

- オンラインランキング機能の実装  
  - PlayFabを用いてスコアの送信・取得、ユーザー名の設定を実装  
  - スコアを昇順で取得し、ランキング表示  

- FEVER状態の演出  
  - 文字オブジェクトを右 → 中央 → 左に移動させるアニメーションを実装  
  - Update処理で文字色を時間経過に応じて変化させ、視覚的な盛り上がりを演出
